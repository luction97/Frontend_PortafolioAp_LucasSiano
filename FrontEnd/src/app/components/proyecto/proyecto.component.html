PROYECTOS

"Wtfood"

Trabajo en cooperativo en la creación de una página web dinámica sobre recetas, con diversas funcionalidades como; registro de usuario, subir y compartir recetas a una comunidad. Me desempeño tanto en backend como frontend, utilizando tecnologías como; Java, jpa, springboot, html, css, bootstrap, MySQL como base de datos de pruebas y postman para testing. Diseño de fondos e iconos utilizando pothoshop. (En progreso)

"Tecnoticias"

Página estática que simula ser una web de noticias sobre tecnología. Proyecto personal de Maquetado con html, css y bootstrap. Deploy con firebase.

Calculadora con Java

Calculadora funcional ejecutable para escritorio, la cual puede activar su diseño a modo oscuro. Es un proyecto realizado en forma autodidacta, utilizando unicamente java como tecnología principal, y el diseño de botones creado con photoshop.
